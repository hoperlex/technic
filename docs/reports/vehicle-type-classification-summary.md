## Source mappings

Однозначные (`direct`) → конечный подтип: Автокраны→`truck_crane`, Кран пневмоколесный→
`pneumatic_tire_crane`, Кран самоходный→`self_propelled_crane`, Легковые автомобили→`passenger_car`,
Тягачи с полуприцепами→`tractor_unit`, Фронтальные погрузчики→`front_loader`, Экскаватор колесный→
`wheeled_excavator`, Экскаваторы гусеничные→`crawler_excavator`, Катки→`road_roller`.

Неоднозначные (`by_model`/`by_registry`, `requires_instance_resolution=true`) → родитель: Вилочные
погрузчики и минипогрузчики→`loaders`, Грузовые малотоннажные автомобили→`road_transport`, Тяжелые
манипуляторы→`kmu_transport`, Бункеровозы, мультилифты→`specialized_transport`, Экскаваторы-погрузчики→
`excavators`.
